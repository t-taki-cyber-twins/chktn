# 修正内容の確認 (Walkthrough)

## 変更概要
`docs/design/wireframe/css/common.css` に新しいボタンクラスを追加しました。

## 変更詳細

### 追加されたクラス
- `.btn-success`: 成功アクション用 (緑)
- `.btn-info`: 情報表示用 (水色)
- `.btn-warning`: 注意喚起用 (黄色)
- `.btn-danger`: 危険アクション用 (赤)
- `.btn-light`: 明るい背景用 (白/グレー)
- `.btn-dark`: 暗い背景用 (黒)
- `.btn-link`: リンク風ボタン
- `.btn-outline-primary`, `.btn-outline-secondary`, `.btn-outline-danger`: アウトラインボタン

## 確認方法
既存のHTMLファイルでこれらのクラスを使用することで、新しいスタイルが適用されることを確認できます。
