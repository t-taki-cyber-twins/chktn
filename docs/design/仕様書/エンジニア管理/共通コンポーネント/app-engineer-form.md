# エンジニア情報入力フォーム (app-engineer-form.js)

## 概要
エンジニアの基本情報、社員情報、スキル、経歴（案件）情報を入力・管理するための包括的なフォームコンポーネントです。
タブ切り替えにより、多岐にわたる情報を整理して入力することができます。

## 機能の目的
- エンジニアに関する詳細な情報を一元管理し、効率的なデータ入力と閲覧を提供する。
- スキルや経歴などの構造化データを直感的に操作できるようにする。

## ユースケース
- **エンジニア新規登録**: ユーザーは、新しいエンジニアの情報を各タブに入力して登録する。
- **エンジニア情報編集**: ユーザーは、既存のエンジニア情報を閲覧・修正する。
- **スキルシートからの情報取り込み**: ユーザーは、スキルシートをアップロードし、AI解析結果（スキル、案件）をフォームに反映させる。

## ユーザ検討事項
- **営業可否**: 営業可否は、マッチングや相手からの面談申込の可否に影響するすごい重要な項目。原則手動とするが、更新漏れを防ぐための機能は必要か検討。
- **スキルシート解析**: スキルシートアップロード機能はありますが、アップロード後の自動解析・入力補助機能（AIによる抽出など）の仕様詳細を確認する必要があります（現状はコンポーネント内にプレースホルダーがある可能性があります）。
- **案件のソート順**: 案件一覧は「稼働終了日 > 稼働開始日 > 面談日時」の降順でソートされますが、この仕様で問題ないか確認が必要です。
- **案件の検索**: 検索できたほうが良いか？

## エンジニアメモから情報抽出・エンジニアメモを生成について
IT人材紹介界隈でよく見る「エンジニアメモ・スキルメモ」的なものから本システムの情報を抽出する、
またはその逆に本システムの情報から「エンジニアメモ・スキルメモ」的なものを生成する機能。

### ユースケース詳細
#### エンジニアメモから情報抽出
- **エンジニアメモ選択**: ユーザーが「エンジニアメモ」をコピペ
  - こんな感じのやつ
  ```
  ***************************************************
  【名　前】： T.K（37歳・男性)
  【最　寄】： 赤羽駅
  【稼　働】： 4月～
  【単　価】： 49万（応相談）
  【スキル】： Java開発2年（設計～テスト）
  　《言　語》　Java17 / Spring Framework / Kotlin
  　《 D　B 》　SQLServer
  　《その他》　Linux, Shell, Ansible
  【並行】　： 並行営業ご了承下さい
  【備　考】：
  ECサイト構築プロジェクトに参画し、フロント～バックの幅広い領域をカバーしました。今後もフルスタック志向です。
  ***************************************************
  ```
- **エンジニアメモ解析**: AIエージェントを用いて「エンジニアメモ」を解析し、構造化されたデータを生成する
- **エンジニアメモ解析結果反映**: AIエージェントから生成された構造化されたデータを、各フォームのフィールドに反映しAIが入力したことが分かるようにする（この段階ではDBにデータを保存していない）
  - エンジニアメモから解析するのはメモから本システムに入れられる情報全て

#### エンジニアメモを生成
- **テンプレートを選択**: あらかじめテンプレート作っておく（メールの変数置換の仕組みとAIでの補完的なものを想定）
- **エンジニアメモ生成**: 上記テンプレートに沿ってAIエージェントを用いて「エンジニアメモ」を生成する
- **エンジニアメモ生成結果反映**: AIエージェントから生成された「エンジニアメモ」を結果に反映する
