# 共通コンポーネント一覧

## 凡例
各共通コンポーネントは、主な使用箇所に応じて以下のフォルダに分類されます：
- **共通コンポーネント**: 複数機能にまたがって広く使用されるコンポーネント
- **エンジニア管理**: エンジニア管理機能で主に使用されるコンポーネント
- **案件管理**: 案件管理機能で主に使用されるコンポーネント
- **メッセージ・面談**: メッセージ・面談機能で主に使用されるコンポーネント
- **テナント・企業・アカウント管理**: テナント・企業・アカウント管理機能で主に使用されるコンポーネント
- **パブリック・トップ・認証**: パブリック・トップ・認証機能で主に使用されるコンポーネント
- **管理者・マスタ管理**: 管理者・マスタ管理機能で主に使用されるコンポーネント

## 分類の詳細

### 共通コンポーネント フォルダ
複数の機能領域にまたがって使用される汎用的なコンポーネント
- `app-header.js` - 全画面で使用
- `app-sidebar.js` - 全画面で使用
- `app-recent-screens.js` - 全画面で使用
- `app-pagination.js` - 全画面で使用
- `app-employee-selector.js` - 複数機能で社員選択に使用
- `app-project-selector.js` - 複数機能で案件選択に使用
- `app-engineer-selector.js` - 複数機能でエンジニア選択に使用
- `app-official-company-search-modal.js` - 複数機能で法人検索に使用

### メッセージ・面談 フォルダ
メッセージと面談に関連する機能のコンポーネント
- `app-engineer-meeting-edit.js` - 面談編集（複数画面から使用されるが面談機能）
- `app-email-preview.js` - メールプレビュー（主に面談で使用）
- `app-message-list.js` - メッセージ一覧
- `app-message-thread-modal.js` - メッセージスレッド詳細
- `app-message-reply.js` - メッセージ返信
- `app-meeting-round-manager.js` - 面談ラウンド管理
- `app-meeting-tool-settings.js` - 面談ツール設定

### エンジニア管理 フォルダ
エンジニア管理機能で使用されるコンポーネント
- `app-engineer-form.js` - エンジニア情報フォーム
- `app-skill-sheet-uploader.js` - スキルシートアップロード
- `app-employee-info-form.js` - 社員情報フォーム
- `app-project-register-modal.js` - 案件登録モーダル（エンジニア編集画面から使用）

### 案件管理 フォルダ
案件管理機能で使用されるコンポーネント
- `app-project-form.js` - 案件情報フォーム
- `app-blacklist-selector.js` - ブラックリスト選択
- `app-mailing-list-selector.js` - メーリングリスト選択（案件・エンジニア両方で使用されるが案件管理に分類）

### テナント・企業・アカウント管理 フォルダ
テナント、企業、アカウント管理機能で使用されるコンポーネント
- `app-company-edit-modal.js` - 会社情報編集
- `app-employee-edit-modal.js` - 社員情報編集
- `app-tenant-edit-modal.js` - テナント情報編集
- `app-account-edit-modal.js` - アカウント情報編集
- `app-plan-selection-modal.js` - プラン選択
- `app-payment-method-modal.js` - 支払方法選択

### 管理者・マスタ管理 フォルダ
管理者・マスタ管理機能で使用されるコンポーネント
- `app-api-integration.js` - API連携設定
