# エンジニア登録 (engineer-register.html)

## 概要
自社の新しいエンジニア（正社員、契約社員、パートナーなど）をシステムに登録するための画面です。
基本情報、スキル情報、経歴書（スキルシート）などを入力・アップロードし、データベースに保存します。

## 機能の目的
- 新規エンジニアの情報をシステムに取り込み、案件マッチングや管理を行える状態にする。
- スキルシートからの情報抽出（将来的な機能）などにより、登録作業を効率化する。

## ユースケース
- **エンジニアの新規登録**: ユーザーは、エンジニアの氏名、所属、スキル、稼働条件などを手動で入力して登録する。
- **スキルシートのアップロード**: ユーザーは、エンジニアの経歴書ファイルをアップロードする。

## EARS 記法による要件定義

### エンジニア情報入力
- **WHEN** ユーザがエンジニア登録画面を開いた際
  - **THEN** システムは空のエンジニア情報入力フォーム（`app-engineer-form`）を表示する

- **WHEN** ユーザが「登録」ボタンをクリックした際
  - **THEN** システムは入力内容をバリデーションする
  - **IF** バリデーションエラーがない場合
    - **THEN** システムはエンジニア情報を保存する
    - **AND** 完了メッセージを表示し、一覧画面または詳細画面へ遷移する

## ユーザ確認事項
- **キャンセルボタン**: JavaScript (`engineer-register.js`) にはキャンセルボタンの処理が含まれていますが、HTML (`engineer-register.html`) のサイドバーアクションには「登録」ボタンしか存在しません。キャンセルボタンの配置（サイドバーか、フォーム下部か）を確認する必要があります。
- **スキルシート解析**: スキルシートアップロード機能はありますが、アップロード後の自動解析・入力補助機能（AIによる抽出など）の仕様詳細を確認する必要があります（現状はコンポーネント内にプレースホルダーがある可能性があります）。
