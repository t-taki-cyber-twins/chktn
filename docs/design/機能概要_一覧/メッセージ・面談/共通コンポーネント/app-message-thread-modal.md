# メッセージスレッド詳細モーダル (app-message-thread-modal.js)

## 概要
選択されたメッセージスレッド内の全てのメッセージを時系列順に表示するモーダルコンポーネントです。一覧画面では表示しきれない長いやり取りや、過去の経緯を確認するために使用されます。

## 機能の目的
- スレッド内の全メッセージを一箇所で閲覧可能にし、コミュニケーションの文脈を把握しやすくする。

## ユースケース
- **過去のやり取りの確認**: ユーザーが面談前や返信前に、これまでのやり取りの全容を確認する。

## EARS 記法による要件定義

### スレッド表示
- **WHEN** システムがスレッドデータを渡してモーダルを開いた際
  - **THEN** システムは親メッセージを先頭に表示する
  - **AND** 続いて子メッセージ（返信）を送信日時の昇順（古い順）で表示する

### モーダル操作
- **WHEN** ユーザーが「閉じる」ボタンまたはオーバーレイをクリックした際
  - **THEN** システムはモーダルを閉じる
