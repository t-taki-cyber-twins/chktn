# 工数見積計画

本ドキュメントは、各画面および共通コンポーネントの開発工数を見積もるための計画書です。
作業を並行して進められるよう、機能グループごとに分割して定義します。

## 見積もり基準

- **単位**: 人日 (day)
- **範囲**: フロントエンド (FE)、バックエンド (BE)、テスト
- **設計工数**: 各フェーズ（FE/BE/Test）の工数に、それぞれの設計工数を含めること。
- **合計工数**: FE + BE + Test の合計値。

## 実行プロセス

1. 各グループの担当者は、対象の仕様書（Markdown）を読み込む。
2. 仕様の複雑度、画面要素数、API数などを考慮し、工数を算出する。
3. `docs/design/仕様書/画面-仕様書対応一覧.md` および `docs/design/仕様書/共通コンポーネント一覧.md` の該当行に工数を記入する。
